Function: int fedisableexcept (int excepts)
Preliminary:
| MT-Safe 
| AS-Safe 
| AC-Safe 
| See POSIX Safety Concepts.


This functions disables traps for each of the exceptions as indicated by
the parameter except.  The individual exceptions are described in
Status bit operations.  Only the specified exceptions are
disabled, the status of the other exceptions is not changed.

The function returns the previous enabled exceptions in case the
operation was successful, -1 otherwise.
